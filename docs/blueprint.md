# **App Name**: Text Echo

## Core Features:

- Display Message: Present a static 'Hello World' message prominently to the user upon loading the page.

## Style Guidelines:

- Primary color: A deep, muted blue-grey (#526166) conveying stability and professionalism.
- Background color: A very light, subtle blue-grey (#EDF0F1) for a clean, unobtrusive canvas.
- Accent color: A refreshing, clear sky blue (#54C1DB) for potential highlights or interactive elements, offering a pleasant contrast.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly legible characteristics, ensuring clarity in the message.
- Should the application expand, use minimalist and crisp line-art icons that complement the clean aesthetic.
- A minimalist, centered layout focusing entirely on the main 'Hello World' message for immediate user attention and simplicity.
- Subtle, quick fade-in animation for the 'Hello World' text upon initial page load to enhance the welcome experience without distraction.
